# NexusPay: Manual Quality Foundation

This repository contains the strategic quality documentation, risk analysis, and manual test engineering I created for the **NexusPay** Fintech ecosystem.

##  Objectives
- Establish a Risk-Based Testing (RBT) framework for global payments.
- Document end-to-end user journeys for Wallet & AI features.
- Validate data integrity through SQL backend testing.

##  Project Context
NexusPay is a multi-currency wallet with an integrated AI Wealth Assistant. Testing focus is placed on **Security**, **Data Accuracy**, and **User Experience**.
